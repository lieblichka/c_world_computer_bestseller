### DOING SOMETHING 

	- Пока что все команды, которые ранее были описаны
	попадали в одну из двух категорий 

	1) DOING SOMETHING

	2) DOING SOMETHING IF do something if something is true


	# doing something 

		- Большинство команд в Си являются инструкциями. Простые
		инструкции - это действия; они что-то выполняют и что-то
		сообщают: переменные, считывание клавиатурного ввода,
		вывод данных на экран

			split_hand(); - простая инструкция 

		- Иногда несколько простых инструкций группируют в
		блочные. Блочные инструкции - это совокупность команд,
		заключенная в фигурные скобки. 

			{
				deal_first_card();
				deal_second_card();
				cards_in_hand = 2; 
			}

	# doing something if something is true

		- Управляющие инструкции, такие как IF, проверяют условие
		прежде чем выполнить код

			if (value_of_hand <= 16)	// true
				hit();
			else						// false
				stand();


		- Если при соблюдении условия нужно выполнить сразу
		несколько действий, используется блочная инструкция

			if (dealer_card == 6) // true
			{
				double_down();
				hit();
			}


### BRACES
ФИГУРНЫЕ СКОБКИ 

	
	- Обычно используется такая конструкция (фигурные скобки), 
	когда внутри несколько инструкциий : 
		
		if (x == 2)
		{
			call_whitehouse();
			sell_oil()
			x = 0;
		}

	- Фигурные скобки можно не использовать, когда инструкция
	всего одна 

		if (x == 2) 
			puts("Делать что-то");


	- Но также не будет ошибкой 

		if (x == 2)
		{
			puts("Делать что-то");
		}
