### QUESTION 1

	- Что означает card_name[0] ?

	# ANSWER 

		- Это первый символ, который набрал пользователь 
		Таким образом, если он ввел 10, card_name[0] будет 
		равен 1

### QUESTION 2
	
	- Вы всегда пишете комментарии с помощью /* */ ?

	# ANSWER 

		- Если компилятор поддерживает стандарт C99, то
		можно использовать комментарий формата // 
		Компилятор воспримет оставшуюся часть такой строки
		как комментарий

		// it is comment 

### QUESTION 3

	- Как узнать, какой стандарт поддерживает мой компилтор ?

	# ANSWER

		- Проверить документацию для компилятора, gcc
		поддерживает все три стандарта: ANSI C, C99 и C11
