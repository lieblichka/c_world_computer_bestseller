### How to run program 

	- Си компилируемый язык. Это значит, что компьютер не
	будет интерпетировать код напрямую. Вместо этого нам 
	нужно конвертировать (или компилировать) исодный код,
	понятный человеку, в машинный код, который может понять
	компьютер


	# Вот как можно скомпилировать и запустить программу 
	с помощью gcc

		1) Сохранить код, представленный на противоположной
		странице в упражнении с магнитиками, в файл пол
		названием cards.c 

		2) Выполните компиляцию, набрав 

			~ gcc cards.c -o  cards 


		3) Запустить набрав:
			
			Windows # cards

			Linux and Mac # ./cards 


### Уголок ботана 

	- Можно скомпилировать и запустить код за один шаг

		gcc zork.c -o zork && ./zork


		# Команда отработает полностью, если при компиляцией
		не возникнет ошибок
